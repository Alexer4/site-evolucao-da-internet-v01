TCP/IP — A Linguagem da Internet

"1983 · O protocolo que uniu todas as redes em uma só"

O TCP/IP é o conjunto de protocolos que permite a comunicação entre computadores em rede. Desenvolvido por Vint Cerf e Bob Kahn nos anos 1970, foi adotado como padrão em 1º de janeiro de 1983 — data considerada o "nascimento oficial" da internet moderna.

🧩 As 4 Camadas do TCP/IP

Clique em cada camada para expandir e ver os protocolos

CAMADA 4 — Aplicação 🖥️
Interface com o usuário. Protocolos como HTTP (web), SMTP (e-mail), FTP (arquivos) e DNS (nomes) operam aqui. É a camada que você "vê".
HTTP HTTPS SMTP FTP DNS SSH
CAMADA 3 — Transporte 📦
Garante que os dados cheguem completos e na ordem certa. O TCP estabelece conexão e confirma recebimento; o UDP é mais rápido mas sem garantia.
TCP UDP
CAMADA 2 — Rede / Internet 🌐
Roteia os pacotes entre redes diferentes. O protocolo IP define endereços e encontra o melhor caminho. Roteadores operam aqui.
IP ICMP ARP RIP OSPF
CAMADA 1 — Interface de Rede 🔌
A camada física e de enlace. Cabos, placas de rede, Wi-Fi. Envia os bits brutos pelo meio físico entre dispositivos na mesma rede.
Ethernet Wi-Fi PPP MAC
